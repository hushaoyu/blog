扩展语法

表格

1、基本用法：使用管道符分割列，使用三个或多个连字符创建列标题

| syntax | description |
| --- | --- |
| header | title |
| name | value |
| name | value |

syntax description
header title
name value
name value

2、内容对齐：在连字符前后加冒号，左边冒号表示左对齐，右边表示右对齐，两侧都有表示居中

| syntax | description | other |
| :--- | :---: | ---: |
| 1 | title | 1 |
| 1 | value | 1 |
| 1 | value | 1 |

syntax description other
1 title 1
1 value 1
1 value 1
